\section{Hiện thức ứng dụng}

\subsection{Tổng quan}
Nhóm sử dụng ngôn ngữ Python cùng với framework FastAPI làm backend và frontend là framework React.js để hiện thực cho web app.

Kiến trúc: hệ thống được xây dựng theo kiến trúc Client-Server kết hợp với mô hình Microservices đơn giản hóa (đóng gói bằng Docker)

Kết nối database: Nhóm sử dụng thư viện pyodbc cùng với ODBC Driver 17/18 for SQL Server để kết nối với database thông qua các lệnh SQL Query.
